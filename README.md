# DartBasicFunctions

# Project Fungsi Dart (Flutter UI)

## Deskripsi

Project ini dibuat untuk memenuhi tugas pemrograman dasar mengenai penggunaan function dalam bahasa Dart.
Aplikasi dibangun menggunakan Flutter dan memiliki tiga fitur utama yaitu menyapa, menghitung luas persegi, dan menghitung harga setelah diskon.

## Fitur Aplikasi

### 1. Menyapa

User memasukkan nama, kemudian sistem akan menampilkan sapaan berdasarkan input tersebut.

### 2. Luas Persegi

User memasukkan nilai sisi, lalu sistem akan menghitung luas persegi menggunakan rumus:
Luas = sisi × sisi

### 3. Diskon

User memasukkan harga dan nilai diskon, kemudian sistem akan menghitung harga akhir setelah diskon.

## Konsep Function

Dalam project ini digunakan beberapa jenis function:

* Function tanpa parameter tanpa return
* Function dengan parameter tanpa return
* Function tanpa parameter dengan return
* Function dengan parameter dengan return

Function digunakan untuk memisahkan logika program agar lebih terstruktur, mudah dibaca, dan mudah dikembangkan.

## Tampilan Aplikasi

Aplikasi menggunakan tampilan berbasis UI dengan Flutter.
Setiap fitur ditempatkan dalam card terpisah sehingga memudahkan user dalam berinteraksi.

## Struktur Project

* lib/main.dart : Berisi seluruh kode program
* pubspec.yaml : Konfigurasi project Flutter
* README.md : Dokumentasi project

## Cara Menjalankan

1. Pastikan Flutter sudah terinstall
2. Buka terminal pada folder project
3. Jalankan perintah berikut:

flutter run

## Kesimpulan

Project ini membantu memahami konsep dasar function serta penerapannya dalam aplikasi sederhana berbasis UI menggunakan Flutter.
